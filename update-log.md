# 更新日誌

#### 版本號格式說明

**v x.x.x pre.yyyy.dddd**

**v:** 表示version的意思

**x.x.x:** 表示 **[大版本號/定期更新版本號/小修改版本號(次數)]**

**pre~:** 代表preview的意思

**~yyyy.dddd:** 代表預發布版開發完成日
